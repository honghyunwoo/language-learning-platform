## 🎯 한국 타겟 재정의

### ❌ Before (글로벌 기준)
- **타겟**: B1 면접 영어 (CEFR)
- **문제**: 한국인은 CEFR 모름, B1이 뭔지 모름

### ✅ After (한국 기준)

**타겟**: **토익스피킹 IH (Intermediate High) 레벨 취업 준비생**

**왜 토익스피킹?**
- 한국 기업 90% 이상 토익스피킹 점수 요구
- IH = 140-160점 (면접 가능 최소 레벨)
- CEFR B1과 거의 동일 레벨
- **한국인에게 익숙한 기준**

**세부 타겟**:
- **1순위**: 외국계 취준생 (20-30대)
  - 토익스피킹 IH-AL 목표
  - 영어 면접 필수
  - 온라인 학습 선호
  - 지불 능력 있음

- **2순위**: 대기업 취준생
  - 토익스피킹 IH 이상 필요
  - 면접에서 영어 질문 나올 수 있음
  - 학원보다 저렴한 대안 찾음

**TAM (Total Addressable Market) 재계산**:
```
한국 취업 준비생: 500K명/년
→ 토익스피킹 응시자: 30% = 150K명
→ IH 목표 (중급): 40% = 60K명
→ 외국계/대기업 지원: 30% = 18K명
→ 온라인 학습 선호: 40% = 7.2K명
→ 지불 의향 40%: 2.9K명
→ 1% 도달: 29명 (현실적!)
```
